# Desafio: API Bancária Assíncrona com FastAPI (Luizalabs)

Projetar e implementar uma API RESTful assíncrona usando FastAPI para gerenciar operações bancárias de depósitos e saques, vinculadas a contas correntes

Aprendizxado com base no módulo "APIs Modernas com FastAPI do Zero ao Deploy", desafio "Modelando o Sistema Bancário em POO com Python" e na resolução do Instrutor.
